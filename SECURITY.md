# Security Policy

## Supported Versions

Security fixes are provided for the latest minor release on the `main` branch.

| Version | Supported |
| --- | --- |
| 0.2.x | Yes |
| < 0.2.0 | No |

## Reporting a Vulnerability

Please open a private security advisory in GitHub or email the maintainers through the support channel in `SUPPORT.md`.

When reporting, include:

- Affected OpenAtoms version
- Reproduction steps and expected vs actual behavior
- Impact assessment (safety, integrity, availability)
- Suggested mitigation if known

We will acknowledge reports within 72 hours and provide remediation guidance or a patch timeline.
